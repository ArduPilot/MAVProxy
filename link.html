MAVProxy

A UAV ground station software package for MAVLink based systems

Link Management

The link module allows the user to add, remove and monitor the status of communications links to the APM. This can be done at any time.

It is useful for UAV systems that have multiple communications links. MAVProxy will automatically read the data from both links, no user intervention is neccessary for link failover.

A link menu is available on the GUI console.

link add

Add an additional communications link. This can be either a network address or serial port. For a serial port, the default baud rate is 57600. Use set baudrate xxx to change baud rate for any susequent link add's.

link add 127.0.0.1:14550
set baudrate 115200
link add COM17

link remove

Remove a communications link. The ID of the link is used to identify the link to remove.

link remove 2

link list

List the active communications links and their ID's

link list
