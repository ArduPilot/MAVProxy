MAVProxy

A UAV ground station software package for MAVLink based systems

Quickstart

In the simplest form, MAVProxy only needs to address of the USB port or network address to connect to.

Over USB

If there is only 1 APM connected, the --master is not required. MAVProxy will autodetect correct port.

Linux:
mavproxy.py --master=/dev/ttyUSB0

Windows:
mavproxy.py --master="com14"

Over Serial/Radio link

If there is only 1 APM connected, the --master is not required. MAVProxy will autodetect correct port.The baud rate will need to match the serial link being used. It is usually 57600 baud.

Linux:
mavproxy.py --master=/dev/ttyUSB0 --baudrate=57600

Windows:
mavproxy.py --master="com14" --baudrate=57600

Over Network

The address to connect to must be your own IP address.

mavproxy.py --master=192.168.1.1:14550

Windows Specific Issues

Windows users may need to add the Python path before using MAVProxy.

setlocal
SET PATH=%PATH%;C:\Python27\Scripts
mavproxy.py --master=192.168.1.4:14550
pause
