MAVProxy

A UAV ground station software package for MAVLink based systems

Development Environment

For developing MAVProxy (such as writing custom modules), a slightly more complex setup is required. The following section will show how this setup is achieved.

This section assumes basic knowledge of setting up development environments, Python and git.

Linux

Install the Python and MAVlink libraries as per the guide. Additionally, install the setuptools package via pip

Use git to download the MAVProxy source: git clone https://github.com/tridge/MAVProxy.git

After making the desired changes, MAVProxy is required to be installed (the modules won't work otherwise). This needs to happen after any changes to the source code. This can be done by:

python setup.py build install --user

MAVProxy can then be run as per normal.

Windows

In Windows, the process is a bit more complicated.

You will need to have Python 2.7, wxPython and pygame installed first.

Next, open up a console in the Python scripts install path (C:\Python27\Scripts or similar) and use pip to install the other packages:
pip install pyinstaller
pip install matplotlib
pip install pyreadline
pip install pyserial
pip install pymavlink
pip install python-dateutil pytz pyparsing six

The final set of prerequisites use pip to install, but need to be manually downloaded. Use pip install [filepath] to install them:

  • numPY
  • Pillow (replaces PIL)
  • OpenCV

Download the MAVProxy source.

After making the desired changes, MAVProxy is required to be compiled and copied into the Python directory (the modules won't work otherwise). This needs to happen after any changes to the source code. This can be done by running the ./MAVProxy/MAVProxyWinUSB.bat file. This will perform the necessary actions and then run MAVProxy.
