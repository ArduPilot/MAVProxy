MAVProxy

A UAV ground station software package for MAVLink based systems

System Commands

There are a number of overall system and miscellaneous commands:

reboot

Reboots the APM.

setup

Goes into the setup (CLI) mode of the APM.

rc

Override a RC (input) channel. This value remains in effect until a value of -1 is set. Uses the form of rc chan value. Use all to set a global value for all RC channels.

rc 1 1000
rc all -1

servo

Override a servo (output) channel. Useful for manually triggering the auxillary channels. Uses the form of servo chan value.

servo 8 1500
