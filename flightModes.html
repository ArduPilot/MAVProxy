MAVProxy

A UAV ground station software package for MAVLink based systems

Flight Modes

Some flight modes can be directly entered on the command line. This will send to appropriate command to the APM.

auto

Enter AUTO mode, starting with the first waypoint.

loiter

Enter LOITER mode around the UAV's current position.

rtl

Enter Return to Launch mode.

manual

Enter manual mode. On Arducopter this is equivalent to STABILIZE mode

fbwa

Enter Fly By Wire A mode. Applicable on fixed wing APM's only.

Arming/Disarming

The APM can be armed and disarmed at any time by using the arm and disarm commands.
